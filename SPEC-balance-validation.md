# SPEC: 출시 전 밸런스 검증 기준 (플레이테스트)

> CPO 트리니티 · 2026-06-17 · 실행: 모피어스(COO) 플레이테스트
> 목적: Go/No-Go P0-③("신규 무과금이 ch18까지 막힘없이") 실빌드 판정.
> sim은 상대균형 근사 — 이건 **실제 플레이 체감** 판정. 무과금 신규 1명을 처음부터 플레이.

## 검증 1 — 신규 진행 곡선 (가장 중요)
| 구간 | 기대 | Pass 기준 | Fail 시 조정 |
|---|---|---|---|
| ch1~8 | 초보 완전 승리 | 강화/가챠 없이 자동으로 전승, 막힘 0 | 적 수/배율 ↓ |
| ch9~17 | 시스템 맛보기 | 가챠·강화·편성 중 1~2개만 써도 통과. 과한 정체 없음 | enemyForChapter 완만화 |
| ch18 진입 | 환생 게이트 | 무과금이 **30분~첫 세션 내 ch18 도달** → "🔄환생 가능" 배너 노출 | 게이트/곡선 조정 |
| ch18→ | 첫 환생 | 환생 후 영구배율로 초반 **눈에 띄게 빨리** 녹음(도파민) | 1.08 미세조정 |

**핵심 Fail 신호**: ch9~17에서 무과금이 벽 느껴 이탈하면 = 출시 차단. (환생 도달 전 이탈 = 루프 자체가 안 돌아감)

## 검증 2 — N등급 60% 드랍 체감
- 가챠 60%가 N(무과금 주력). **Pass**: N 위주로 받아도 편성·진행 가능 + R/SR 받을 때 "오 좋은거!" 차별 체감.
- **Fail**: N만 잔뜩 → "쓰레기만 나온다" 좌절 / 또는 N이 너무 약해 무의미. → N 스탯(mul 1.0) 또는 드랍률 조정.

## 검증 3 — 환생 루프 (1~3회)
- **Pass**: 환생 1·2·3회차가 매번 직전 벽 +2~5챕 전진, 재돌파 즉시 체감. 에테르 상점 노드 1~2개 살 만큼 수급.
- **Fail**: 환생해도 전진 없음(stall) 또는 1~2회로 게임 트리비얼(폭주). → 노드 배율/에테르 곡선.

## 검증 4 — 가챠·천장 체감
- **Pass**: 천장 "N회 남음" 가시적, 첫 SSR 도달이 좌절 아닌 기대. 첫구매2배 명확.
- **Fail**: 천장까지 너무 멀게 느껴짐 / SSR이 안 나와 답답. → 천장/소프트피티.

## 검증 5 — 전투 템포 (도파민)
- **Pass**: 한 판 길이 적절(너무 길어 지루/짧아 허무 아님), 자동사냥 방치 진행 매끄러움, 궁극기·이펙트 손맛.
- **Fail**: 전투 늘어짐, 자동 멈춤, 피드백 밋밋.

## 판정 방법
1. localStorage 초기화(완전 신규)로 시작.
2. 무과금 가정(결제 0)으로 ch1→ch18→첫환생까지 실제 플레이.
3. 각 검증 Pass/Fail + 막힌 지점·소요시간 기록.
4. P0-③은 **검증1이 Pass여야 출시 Go.** 나머지는 Fail이면 P1 조정.
