# 🎵 배경음악 폴더

`audio/bgm.mp3` 를 넣으면 합성 BGM 대신 **실제 음원**이 루프 재생된다. 없으면 자동으로 합성 딥하우스 폴백.

## ⚖️ 반드시 로열티프리(상업 라이선스) 음원만
저작권 있는 음악 절대 금지. 무료 상업 사용 가능한 곳:
- **Pixabay Music** (pixabay.com/music) — 가입 없이 무료, 상업 OK
- **Free Music Archive** (freemusicarchive.org) — CC0/CC-BY 필터
- **Incompetech** (incompetech.com) — CC-BY (크레딧 표기)
- **YouTube Audio Library** — 다운로드 무료, 라이선스 확인

검색어: `deep house loop`, `lofi house`, `chill house seamless loop`

## 권장
- 포맷 `.mp3` (모바일 호환), 파일명 정확히 `bgm.mp3`
- **끊김 없는 루프(seamless loop)** 음원 권장 — 무한 반복돼도 어색하지 않게
- 1~3분, 용량 작게(텔레그램 모바일 로딩). 볼륨은 코드에서 0.45로 조절됨
- CC-BY면 게임 크레딧/설정에 출처 한 줄 표기

넣고 `git push` → 1분 뒤 라이브. 파일 없으면 합성 딥하우스가 계속 재생.
