## 1. 다형 화폐 (Polymorphic Currency) — 잔액 찌꺼기 재충전
**왜**: 화폐가 1종이면 정확히 다 쓰지만, 비호환 다종이면 "230젬 남았는데 부족" → 재충전. 中 게임 표준.
**p1 현황**: gold/gems/soul/ether 이미 4종(game.js:604). → **이벤트 전용 화폐 1종 추가**(축제때만 유효, 만료).
**설계**: 한정 배너는 "스타더스트"로만 구매 → 스타더스트는 젬으로만 충전(끝수 안 맞게 패키지 설계: 100·550·1200). 잔액은 다음 축제까지 동결.
**⚠️ 미꾸라지**: 각 화폐 환율·용도 **명시 표기**(오인 금지=misleading 회피). 만료 전 명확 고지. 현금 환불경로 유지.

## 2. 재출시 고가화 (Remedy Consumption) — 놓친 후회 2차 수확
**왜**: 한정 놓친 유저의 후회를 더 비싸게 재판매. 中 corpus 전용 신종.
**설계**: 지난 한정 SSR/스킨을 **"각성 재림(Awakened Encore)"** 으로 재출시 — 단, +15~25% 가격 또는 추가 재료 요구 + "다시는 안 옴" 표기(실제 회전 주기 명확).
**p2**: 인도신화 = 윤회/재림 서사 천연 정합 — "신이 한 번 더 강림" 스토리로 재판매가 *자연스러움*.
**⚠️**: "다시 안 옴" 표기는 **실제와 일치**해야 함(거짓=3배 징벌 KR). 회전 스케줄 내부고정.

## 3. 끝없는 미끼 (Never-Ending Lure) — 누적지출 VIP 사다리 ★P0
**왜**: 첫충전 할인(있음) + **모든 결제 합산 → 영구 VIP 등급 해금**. "다음 등급" 진행감이 무한 sink.
**p1 현황**: 첫구매 2배·VIP/울트라팩 있음(SPEC-funnel 다리2). → **등급화가 빠진 핵심**.
**설계**: 총 결제액 누적 → VIP 1~15 계단. 각 등급 영구혜택(골드+%, 패스 할인, 전용 스킨, 일일 무료뽑기). "다음 등급까지 ₩X" 진행바.
**⚠️**: 등급 혜택 정확 표기. pay-to-win 과시는 OK이나 미성년 한도 준수.

## 4. 과부하 할인미로 (Overloading) — 인지부하로 정가 비교 마비
**왜**: 할인 구조를 일부러 복잡하게 → 유저가 "이득인지" 계산 못 하고 구매.
**설계**: "왕관코인" 모아 5개 패키지에 **랜덤 할인율(상한 캡)** 적용 — 어디에 쓸지 고민시키며 정가 anchor 흐림.
**우선 낮음(P2)**: 단독 효과보다 보조. p1 먼저 검증 후.

## 5. 서사적 의무 (Narrative Obligation) — 결제를 스토리상 의무로 ★p2 킬러
**왜**: NPC가 "널 믿어"식 감정연계로 결제·플레이를 *서사적 의무*로 프레이밍. 죄책감 레버.
**p1**: 보스위엄/군단 NPC가 "군단이 너 없인 무너진다" — 한정 지원 결제 유도 대사.
**p2 ★**: 인도신화 신(神)이 **유저 이름 호명** → "나의 화신이여, 다르마를 완성하라" → 결제가 *신앙적 의무*. 집단주의 문화 정합(논문: JP/CN 집단주의서 효과↑).
**⚠️**: 죄책감 과도·취약계층(미성년) 타겟 금지. "재미있는 운명" 프레이밍 유지. 강요 아닌 *초대* 톤.

## 6. AI 실시간 개인화 LiveOps — 유저별 동적 오퍼 ★서버측
**왜**: 2025 탑게임 매출 60%가 LiveOps. 승률·시간대·접속·잔액 보고 이벤트/오퍼를 **유저별** 조정.
**설계**: analytics-worker 확장 — 이탈징후(접속간격↑) 시 컴백오퍼, 연패 시 위로팩(near-miss), 고잔액·고접속 시 고가팩 노출. 시간대별 푸시.
**구현 의존**: legion-analytics 워커 + KV 세그먼트. Oracle(CDO) 협업.
**⚠️**: 가격차별은 **표시오퍼 차등**까지(동일상품 다른가격=차별금지 리스크). prominent 표시. 미성년 제외.

## 증폭기 (저비용 보조)
- **S++ 희소등급 시각위계**: B→A→S→S++ 등급 컬러/이펙트 위계 — 희소=럭셔리 지각.
- **NPC 가챠 권유**: 뽑기 화면 NPC "운명이 부른다" 대사(near-miss 후 "한 번 더").
- **Kawaii/신성 미감**: p1 영웅·p2 신 일러 매력도 = 조작 증폭(논문 DP Enhancer).
- **Sycophant**: 인게임 보상으로 X/텔레그램 팔로우 유도(이미 레퍼럴 일부) → 채널 확장.
