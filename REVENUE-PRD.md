# PRD: LEGION 수익 보정 — 출시 직전 머니타이제이션 재설계

> 작성: 2026-06-16 · 작성자: CPO(베라 🎯) · 상태: 검토 · 적용 데드라인: 결제연동 2일 전

## 🎯 핵심 결론 (1줄)
**신뢰 깨는 P0 버그(10연 가격 라벨) 먼저 막고, 첫구매2배·배틀패스·천장 가시화 3개로 "결제 진입로"를 깔아라. 가격은 monthly·weekly·ultra가 업계 대비 비싸 전환을 죽이고 있으니 출시 전 조정 — 단 다크패턴은 하나도 넣지 않는다.**

---

## 1. North Star + 보조 + 가드레일 지표

**North Star: 주간 코어루프 완주 결제유저 비율** (= 주 1회+ 전투→환생 루프 돌면서 1회+ 결제한 유저 / WAU).
- 목표: 출시 D30까지 **8%** (가챠 결제전환 통상 2~5%, 코어루프 결합이라 상향).

**보조(선행) 지표 3개**
1. **첫구매 전환율** (신규→첫 결제): 목표 D7 **6%+**.
2. **배틀패스 유료 전환율** (시즌당): 목표 시즌1 **10%+**.
3. **가챠 천장 도달 후 재가챠율**: 목표 **40%+**.

**가드레일 지표 (나빠지면 즉시 롤백)**
- **D7 리텐션**: 베이스라인 대비 **-3%p 이하 하락 금지**.
- **환불·이탈 문의율**: **2% 이하**.
- **무과금 코어루프 완주 가능성**: 무과금도 환생 1회까지 도달 중앙시간 **48시간 이내**.

## 2. 우선순위 P0/P1/P2 (RICE 근거)

**P0 — 결제연동 전 2일 내 필수 (신뢰·법규)**
- **10연 가격 라벨 통일** (💎30→80젬 코드 일치) — 신뢰 직결, 라벨 사기 = 환불·악평 + 한국 표시광고법 리스크.
- **가챠 확률·하드천장(10) 화면 표기** — 한국 확률형아이템 정보공개 의무화(2024 게임산업법). 선택 아님.
- **첫구매2배 팩** — 결제 진입장벽 최저. 단일 최강 ROI 레버.

**P1 — 출시 후 1~2주 (단기 매출)**
- **가격 재조정**(monthly/weekly/ultra/첫구매) — 비싼 가격이 전환을 죽임.
- **천장 진행바 + 위시리스트** — "조금만 더" 심리 + SSR 지정.
- **출석 뽑기티켓** (7일차 10연티켓) — 리텐션×결제동기.

**P2 — 다음 시즌**
- **배틀패스(시즌제)** — 최대 LTV 레버. 시즌1은 간소판.

**🚫 지금 안 할 것**: PvP·길드전·랭킹 / 한정 픽업·페이트 / 광고 보상형 / VIP 등급 세분화.

## 4. 신규 수익장치 PRD

**A. 첫구매2배 팩 (P0)** — 각 젬팩 *최초 1회* 젬 2배 지급(계정당 팩별 1회, "2배" 뱃지). 성공: 첫구매 전환 D7 6%+. 가드레일: 진짜 추가지급(정가 인상 트릭 금지).

**B. 배틀패스 (P2, 시즌1 간소판)** — 시즌 4주, 30티어. 무료트랙(골드·강화석·소량젬) + 유료트랙(★300, SSR조각·젬·전용 칭호/프레임). 데일리미션 XP. 초반 즉시보상→후반 SSR조각. 무료트랙도 환생 1회분 보장. 성공: 유료전환 10%+.

**C. 가챠 가시화 (P0 확률표기 / P1 위시리스트)** — ①확률표(N60/R28/SR10/SSR2) ②하드천장 진행바(n/10) ③위시리스트(SSR 1종 지정→천장 시 확정). 성공: 천장도달 후 재가챠 40%+. 가드레일: 천장·확률 코드값 100% 일치.

**D. 출석 뽑기티켓 (P1)** — 7일 출석에 단챱 티켓 분산 + 7일차 10연 티켓 1장(화폐 무소모). 성공: D7 리텐션 +3%p. 가드레일: 티켓 SSR 확률 동일(추가 어드밴티지 없음).

## 5. ⚠️ 윤리 가드레일 (수익보다 유저 신뢰)
- **거짓표기 절대금지**: 모든 가격·확률·천장 코드값 일치(10연 버그 = 신뢰 사망).
- **다크패턴 회피선**: 가짜 카운트다운·미끼가격·강제 연속결제·아동 표적 압박 전부 금지.
- **P2W 상한**: 무과금도 코어루프 완주 가능(가드레일 감시). 결제는 가속이지 해금조건 아님.
- **한국 법규**: 확률 의무공개·청소년 결제한도·표시광고법. → **법무 체크는 군주 결정(에스컬레이션).**
