# SPEC: 조합 버프 가시화 (Buff Legibility) — 출시 전 · 저비용

> 작성: 2026-06-16 · CPO(트리니티) · 상태: 구현 스펙(모피어스 COO) · **출시 전 포함(쌈)**
> 원칙: **신규 시스템 0. 이미 있는 `squadSynergy()` 데이터를 *보이게·갖고싶게*만 한다.**

## 0. 왜 (한 줄)
조합 버프가 4겹(진영/다양성/HostWeave/Militia)이나 도는데 **유저는 왜 세진지 모름** → 안 보이는 버프는 행동(편성·가챠)을 못 만든다. **현재 버프 = "지금 가진 것"만 표시(game.js:472 updateBattleCombo).** 빠진 건 ①다음 티어 유도 ②가챠에서 "뽑으면 잠금해제" 미끼. 이 둘만 채우면 **수집·과금 동기**가 생긴다.

## 1. 성공 기준 (측정)
- 편성 화면에서 유저가 "한 칸 더 채우면 버프"를 인지(→ 편성 변경·가챠 행동 ↑).
- 가챠 직후 시너지 잠금해제 노출 → 추가 뽑기 전환 ↑(수익화 벽 보조).
- 가드레일: 정보 과부하 금지(한 화면 버프 칩 ≤ 4개, 다음 티어 힌트 1개만) / 신규 ch≤5 구간엔 미니멀(압도 금지).

## 2. 범위 — 3서피스 (A·B 필수, C 옵션)

### A. 편성/전투 화면 — "다음 티어" 유도 (기존 #battle-combo 확장)
- **현재**: `updateBattleCombo()`가 보유 보너스만 칩으로 나열(game.js:472-487).
- **추가**: 보유 칩 뒤에 **다음 티어 진척 힌트 1개**(가장 가까운 미달 보너스).
  - 데이터: `squadSynergy()`의 `fac{}` 카운트 재사용. 진영 n=2→"+1로 +18%", n=3→"+1로 +30%". 아키타입 distinct도 동일.
  - 카피 예: `🔒 불꽃단 +1체 → 공격 +30%` (회색 점선 칩, 보유 칩과 시각 구분).
  - 한 번에 **가장 임박한 1개만**(전체 나열 ❌ — 과부하 방지).

### B. 가챠 결과 — "잠금해제" 미끼 (★수익화 레버)
- **트리거**: 뽑은 유닛이 *현재 편성 기준* 시너지 티어를 올리거나 **임박(n−1)** 시켰을 때.
- **표시**: 결과 카드 하단 1줄 플래시 — `🏷️ 불꽃단 3/4 — 1체 더면 공격 +30%!` 또는 달성 시 `✨ 다양성 시너지 +20% 잠금해제!`.
- **목적**: 뽑기를 *목표*로 전환 — "한 끗 남았네 → 한 번 더." (다크패턴 아님: 실제 보유 시너지 사실 표시.)
- 데이터: `grantUnit()` 직후(game.js:1447 근처) `squadSynergy()` 호출해 before/after 비교.

### C. 도감/로스터 — 진영 세트 목표 (옵션·출시 후 가능)
- 도감에 진영별 보유/목표 바: `불꽃단 ▮▮▮▯ 3/4 → +30%`. 컬렉션을 *체크리스트*로.
- D-5 빡빡하면 **컷** → v2 동승.

## 3. 카피 원칙
- 숫자 명확(+30% 등 실제 효과), 진영명 표시, 이모지 1개로 종류 구분(🏷️진영/🔀다양성/🌊Weave).
- "잠금/임박"은 **회색·점선**, "발동/보유"는 **컬러·채움** — 한눈에 가진 것 vs 가질 것.

## 4. 비범위 (이번에 안 함)
- ❌ 시너지 *수치/종류* 변경(밸런스 손 안 댐 — 표시만).
- ❌ 신규 버프·페어버프(= v2 트랙).
- ❌ HostWeave/Militia 내부 로직 변경.
- ❌ 5속성 관련 일체(= v2).

## 5. 수용 기준 (Acceptance)
1. 편성에 진영 2체 → "🔒 +1체 → +18%" 점선 칩 1개 노출. 3체 → "+30%"로 갱신. 4체 → 점선 사라지고 보유 칩만.
2. 가챠로 진영 3→4체 달성 유닛 뽑으면 결과 카드에 "✨ +30% 잠금해제" 1.5초 플래시.
3. ch≤5 구간은 점선 힌트 숨김(미니멀).
4. 기존 보너스 수치·전투 결과는 **불변**(표시 레이어만 추가).

## 6. 컷라인 (D-5)
- **필수**: A(다음 티어 힌트) + B(가챠 잠금해제 플래시). 둘 다 `squadSynergy()` 재사용 = 반나절급.
- **옵션→컷 가능**: C(도감 세트바) → 시간 남으면, 아니면 v2.
