# 경쟁 게임 분석 → 다이달로스 추가 요소 (Trinity CPO, 2026-07-09)

승자들 서치·분해 → p1에 없는 뽑을 것. 근거: AFK Journey/Arena·Legend of Mushroom·Idle Heroes·Catizen·Notcoin·Coin Master.

## 승자별 핵심 (무엇이 리텐션/과금을 만드나)
- **AFK Journey** ($150M·16M DL): **공유 레벨링**(영웅 5명 키우면 전군 파워 — F2P 그라인드 벽 제거) · 6파벌 시너지+스토리 · **시즌 업데이트**(신영웅/이벤트=신선도).
- **Legend of Mushroom**: 단일캐릭 깊은 룬 시스템 · 데일리+잦은 이벤트.
- **Idle Heroes**: **데일리 태스크 ~10분 완결**(습관) · 관대한 가챠 · 커뮤니티 이벤트.
- **Coin Master** ($3B): **스핀=에너지**(5개/50분, 다 스핀으로 복귀) · **소셜 습격/공격 PvP**(남의 마을 털기=관계+경쟁, 단기+장기 리텐션 동시) · **실드 3개 캡**(복귀장치) · 타임한정 번들 FOMO.
- **Telegram(Catizen 36M·Notcoin 35M)**: 단순루프+**초대 배수+데일리+마일스톤** · **바이럴 에너지가 엔진**. ⚠️ **토큰/에어드랍 의존 = 사후 리텐션 붕괴**(100M→13M). → in-game 가치가 정답(우리 방향 맞음).

## 🎯 다이달로스에 추가할 것 (우선순위)
**A1. 비동기 PvP 습격/침공 (Coin Master $3B 엔진) — 최대 레버, 지금 전무**
아레나 placeholder → 다른 군단 습격해 골드 약탈 + 복수 + 실드. 관계+경쟁+지위. 방치형+소셜 리텐션의 핵심. **가장 큰 미싱 피스.** (Phase2급 규모 but 최우선 후보)

**A2. 공유/파벌 레벨링 (AFK Journey) — 그라인드 벽 제거**
현 per-char 레벨링은 F2P 벽. 파벌/공유 레벨로 전군 파워 → 로스터 더 씀 = 만족+리텐션. 밸런스 재설계 필요(수학자/Oracle 검증).

**A3. 시즌 케이던스 (AFK Journey seasons + persephone F5)**
롤링 시즌: 신 SSR + 시즌 이벤트 + 마감 명예. 신선도=장기 리텐션. persephone F5 페스티벌과 통합.

**A4. 데일리 10분 완결 체크리스트 (Idle Heroes)**
데일리 미션을 "10분 안에 딱 끝나는" 명확한 체크리스트로 타이트하게 → 습관 형성. (현 데일리 있음 → 완결감·시간 압축.)

## 안 할 것 (경쟁분석 교훈)
