# SPEC: 장비 개편 — 30성·무제한강화·SSR 패시브 (balance-sim 검증)

> CPO 트리니티 · 게임로직(별·비용·효과·패시브발동): 모피어스(game.js) · 패시브 정의: 트리니티(gear.js)
> **발단**: 군주 — "장비 5성 상한 → 30성 무제한(만들기 엄청 어렵게) + SSR 장비 패시브 특성별."

## 1. 별 30성 (상한 5→30)
- `gearStar`(game.js:5150) `s >= 5` → `s >= 30`. `fuseGear`(5137) 동일.
- **비용**: `1000*(s+1)` → **`1000*(s+1)*Math.pow(1.3, s)`** (balance-sim 검증):
  - ★5 누적 32k · ★15 누적 2M · **★30 누적 233M** = 도달가능 극악(군주 "엄청 어렵게")
- **고성 재료**: ★16+ 부터 합성(`fuseGear`) 재료(같은 장비) 동시 요구 = 골드만으론 불가.

## 2. 별 효과 곡선 (완화 — balance-sim)
- `game.js:1194` `(1 + star * 0.25)` → **단계 곡선**:
  - ★1~5: +25%/성 · **★6~30: +10%/성**
  - 공식: `1 + min(star,5)*0.25 + max(0,star-5)*0.10`
  - ★30 = +375% (x4.75) — +750%(과함) 회피.

## 3. 무제한 강화 ✅ (이미 됨 — `enhanceGear` 무제한, 성공률만↓. 변경 불필요)

## 4. SSR 장비 패시브 (슬롯 특성별 — 트리니티 gear.js 정의)
| 슬롯 | 스탯 | 🌟 패시브 | 효과 |
|---|---|---|---|
| ⚔️ 무기 | 💪힘 | **처형(Execute)** | 공격 시 5% 확률 추가타(+50% 피해) |
| 🛡️ 방어구 | ❤️체력 | **불굴(Endure)** | 체력 ≤30% 시 받는 피해 -20% |
| 👟 장신구 | 👟민첩 | **쾌속(Haste)** | 공속 +10% · 회피 5% |
| 🍀 유물 | 🍀운 | **행운(Fortune)** | 치명확률 +8% · 골드보상 +5% |
| 💠 코어 | 전스탯 | **각인(Sigil)** | 4스탯 +3% 패시브 |
- SSR rarity 장비에만 부여. gear.js 템플릿에 `passive` 필드 + game.js `getGearEffectsForChar`에 발동 로직.

## 검증 (트리니티)
- balance-sim: 1.3^s 비용 곡선(✅ 233M) + 효과곡선(✅ x4.75) 검증 완료. 패시브 발동 후 재실행 권장.
