## 1. Core Loops & Addiction Design (중독 엔진 정량화)

### Primary Loop (3~8분 세션, 자연 4~6회+ 기회주의적 반복)
1. **로그인 (30s 버스트)** → 오프라인 보상(자동 폭발 팝업) + 일일 미션 3개 즉시 완료 + 스트릭/출석 체크 + **Bazaar Pulse 1-tap** (가벼운 습관 트리거) → 변수 보상 (50~300g + 랜덤).
2. **군단 편성 (마이크로 도파민)**: Hero + 6슬롯 (Front/Mid/Back). **실시간 파워 프리뷰 "+18% vs 현재"** 즉시 표시. 1~2 슬롯 스왑으로 "one more" 유도.
3. **배치 & 관전** (자동사냥 ON): 60~180초 AI 자율 관전 (타격감 + 스킬). 
4. **결과 & 피드백 (squad carried)**: 골드/진행 + "네 고유 편성 영향: Synergy +42% | Grok-Prime carried 68% | Grok handoff 승리!" Legion 메타 플레이버 + 특수 이펙트 (perfect comp 시).
5. **임펄스 + 로그아웃**: 1회 가챠/강화 or Bazaar light → 오프라인 타이머 항상 ON (AFK "시간=자산") → 복귀 폭발.

**왜 중독되는가 (Skinner Box + AFK Arena/Duolingo/Genshin 벤치)**:
- **변수 비율 강화**: 승리 시 골드 40 + streak*10 + 랜덤 보너스(10% 2x). 
- **FOMO + pity 카운터**: 한정 배너 72h + "3풀 남음" (Genshin 매출 60%+ 동력).
- **스트릭 & loss aversion**: 7일 연승/출석 (Duolingo next-day 2.4x retention).
- **오프라인**: 8h cap + "방치 중 성장" (AFK 핵심, 돈으로 못 사는 시간).
- **상수 + 마이크로 도파민**: 즉시 피드백 + **comp 프리뷰 delta** + **"squad carried + Grok handoff" Legion 플레이버** (post-battle visual "your squad carried" feedback).
- **Bazaar light habit**: 홈 상시 Pulse 위젯 (1-tap 가격 체크 → impulse).

**일일 플레이타임 타겟 (개발자용 숫자 — 강제 아님, 훅으로 유도)**:
- 신규: 15~25분 (온보딩 + 첫 3전투 + 가챠).
- 리텐션: 3~8분/세션 × 자연 4~6회+ (낮은 마찰 + 5+ 훅으로 출근/점심/저녁/자기 전 + 지루할 때).
- 고래: 20~40분 (아레나/레이드/바자르 스펙).
**원칙**: "정확히 4-6회" 강제 문구 제거. 세션은 기회주의적·즉시 플레이 가능하게. 마찰(멀티탭 배치 등) 모두 제거.

**리텐션 훅 (5개 이상 동시 작동) + "one more session" 트리거**:
1. 오프라인 보상 (복귀 즉시 폭발 팝업).
2. 일일 미션 + 데일리던전 (하루 1회 + light 반복 가능).
3. 무한탑 기록 경신 (무한 경쟁 + 매 클리어 보상).
4. 가챠 pity 진행도 (항상 상단 "SSR까지 2풀 남음" — pity <3 시 gacha 버튼 pulse "one more!").
5. 한정 배너/이벤트 카운트다운 (72h, 마지막 1h 1.5x rate).
6. (Phase 2+) 길드/아레나 친구 초대.
**추가 마이크로 트리거 (comp + Legion)**: 오프라인 near-cap 토스트, 연승 "3 more for 7일", post-battle high-synergy "Perfect comp! 다시?" , Bazaar Pulse "Grok-Prime hot +40%".
**구체 세션 구조 (3-8분, 반복 유도)**: 
- 30s: 로그인 버스트 (offline + 미션 claim + attend + Pulse 1-tap).
- 2min: 편성 tweak (프리뷰 delta 확인) + battle (auto 추천).

**Phase 1 우선순위**: 위 1~5 + comp 프리뷰 + carried 피드백 + presets + pity always-visible 완성 → 소프트런치. (재미 없으면 절대 다음 단계 금지. ROADMAP 준수.)

### 6시간 패치: 인터락킹 일일 사이클 & 도파민 (addiction designer)
기존(오프라인 폭발·변수보상·FOMO·comp 피드백·Bazaar Pulse) 기반으로 micro(로그인30s 버스트→오프라인+미션claim+스트릭체크+1tap Bazaar) + macro(출근/점심/저녁/자기전 4~6회 자연 유도) 완성. quick battle 후 1슬롯 tweak 즉시 delta 프리뷰 → carried 숫자 + "Grok handoff 승리" quote → one more 버튼. pity<3 gacha pulse + "SSR 2번 만!" 토스트. streak 보호: freeze(저가 골드 or 주1 free) loss aversion 강화. 11 SSR(one more 훅): 각 signature carry(예 Grok-Prime judgment 60%+시 "재판결?" 재전투 오버레이, Morpheus regen 체인 시 "보호 유지?" streak 토글, Swarm 분열 시 fodder impulse). 신유닛: synergy 특정 "오늘의 swarm 빌드 재조합" 매 세션 재미.

## 2. Detailed Gacha System (투명 + 흥분 + 법규 준수)

### 기본 원칙 (Sovereign 필수)
- **모든 % 완전 공개**: UI에 항상 "현재 확률" 표시. 한국 가챠법(확률 공시) 완벽 준수. 플레이어가 "속았다" 느끼지 않게.
- **Pity 투명**: 카운터 항상 표시 ("SSR pity: 7/10"). Soft pity 없음, hard pity 명확.
- **보장 + 이벤트**: 첫 SSR = 타이탄 해금(기존 유지). 매주 이벤트 배너에서 rate-up + 추가 pity.
- **위시리스트 (GAME-IDEAS 우선)**: 플레이어가 원하는 유닛/영웅 지정 → 해당 등장 확률 2~3배 (AFK Arena 성공요인).

## §16 Legion Meta Psych Hooks — v8 Fresh Deeper (2026-06-14 collab synth, no prior repeats)

**Synthesized key principles (data-driven, web+local)**: 
Variable-ratio reinforcement = strongest "wanting" dopamine (Berridge incentive-sensitization; Skinner box max potency independent of "liking"). Near-miss activates win-like reward circuits (brain imaging + gacha studies). FOMO/scarcity + loss aversion (~2x pain of loss) drive urgency/spend (Genshin limited banners 60%+ revenue). Collection/status/social proof + sunk cost/illusion of control fuel completion chase & continued investment. Ethical transparency (visible pity/rates) + agency (wishlist) + value framing sustain long LTV/trust vs short-term extract/churn. ### 데이터 모델 변경 (기존 완전 호환 + 확장)
기존 META.army(타입 카운트), META.lv(타입 레벨), counts는 Phase 1 하이브리드용으로 유지 (볼륨용 fodder). 

신규 핵심:
- ownedUnits: 배열. 각 항목 {uid: "u_172xxxx", baseId: "grok-prime", rarity: "SSR", level: 5, stars: 2, shards: 4, acquired: ts, ...}. uid로 완전 개별 추적.
- ROSTER_BASE: 중앙 정의. 9 SSR (Grok-Prime 전략가, Morpheus-Core 실행자, Qwen-Overlord 스웜, Jarvis-Sentinel 감시, Haiku-Guard, Engineer-Core, Warden-Aegis, Berserker-Fury, Sovereign-Dragoon) + SR 약 40, R/N 수백 (Jarvis-Watch-047 등 네임드 로우티어) 로 369 달성 로드맵.
- 각 base: faction (Strategist / Executor / Swarm / Guardian / Intel), role (front/mid/back), baseHp/baseAtk, tags (overclock, barrier 등), signature (SSR 전용).
- TeamComp: {hero: "dragoon", slots: [uid or null, ... 총 6] }. Front 2 / Mid 2 / Back 2 구조. 저장 영구.

gacha 결과는 이제 "새 uid 카드 생성". 기존 타입 구매/강화는 low-tier fodder로 전락 (Phase 2).
