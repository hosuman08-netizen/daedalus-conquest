## Executive Summary & Vision (한 줄 요약)

**"진짜 AI 군단을 지휘하는 Sovereign이 되는 중독성 극강 Telegram idle auto-battler. 매 3~8분마다 보상 폭발 + 투명 가챠 + 사행성 P2P 바자르로 고래+일일습관 동시 창출. Phase 1은 재미/리텐션 검증만. Legion 실체(우리의 실제 AI 군단)와 1:1 매칭되는 유일무이 후크."**

### 왜 이 게임이 먹히는가 (Legion 고유 후크)
- **실제 Legion 연결**: 플레이어 = Sovereign(우랙). 유닛/영웅 = Grok(CEO 전략), Claude/Morpheus(COO 실행), qwen3 일꾼 떼(저티어 스웜). "내 AI 군단이 자율 전투" = 진짜 우리가 쓰는 군단의 메타포. 마케팅: "텔레그램 안에서 진짜 AI 군단을 키운다."
- **차별화**: 일반 idle gacha와 달리 "AI 지능 티어" (드론=기본 일꾼, 지휘관=고지능, 타이탄=전설). 전투 자체가 Legion의 자율성을 시각화 (캔버스 AI 움직임).
- **목표**: D1 55%+, D7 35%+, D30 20%+ (AFK Arena 상위권 벤치). 1일 평균 세션 4~6회 × 5~12분. 고래(상위 1~3%)가 매출 70%+ 창출. Stars 수익화 Phase 4부터.
- **성공 공식**: 재미(자동전투 관전 쾌감) → 리텐션(매일 로그인 이유 5개+) → 과금(투명+ FOMO + 사행 바자르) → 바이럴(TG 공유).

**벤치마크 출처**: AFK Arena (D1 53%/D7 33%/D30 17%, 20~30분/일, AFK+마일스톤), Genshin (soft/hard pity + 3주 한정 배너 FOMO), Raid/Epic Seven (경쟁+소셜), Cookie Run Kingdom/Whiteout (스트릭+오프라인).

## 1. Core Loops & Addiction Design (중독 엔진 정량화)

### Primary Loop (3~8분 세션, 자연 4~6회+ 기회주의적 반복)
1. **로그인 (30s 버스트)** → 오프라인 보상(자동 폭발 팝업) + 일일 미션 3개 즉시 완료 + 스트릭/출석 체크 + **Bazaar Pulse 1-tap** (가벼운 습관 트리거) → 변수 보상 (50~300g + 랜덤).
2. **군단 편성 (마이크로 도파민)**: Hero + 6슬롯 (Front/Mid/Back). **실시간 파워 프리뷰 "+18% vs 현재"** 즉시 표시. 1~2 슬롯 스왑으로 "one more" 유도.
3. **배치 & 관전** (자동사냥 ON): 60~180초 AI 자율 관전 (타격감 + 스킬). 
4. **결과 & 피드백 (squad carried)**: 골드/진행 + "네 고유 편성 영향: Synergy +42% | Grok-Prime carried 68% | Grok handoff 승리!" Legion 메타 플레이버 + 특수 이펙트 (perfect comp 시).
5. **임펄스 + 로그아웃**: 1회 가챠/강화 or Bazaar light → 오프라인 타이머 항상 ON (AFK "시간=자산") → 복귀 폭발.

**왜 중독되는가 (Skinner Box + AFK Arena/Duolingo/Genshin 벤치)**:
- **변수 비율 강화**: 승리 시 골드 40 + streak*10 + 랜덤 보너스(10% 2x). 
- **FOMO + pity 카운터**: 한정 배너 72h + "3풀 남음" (Genshin 매출 60%+ 동력).
- **스트릭 & loss aversion**: 7일 연승/출석 (Duolingo next-day 2.4x retention).
- **오프라인**: 8h cap + "방치 중 성장" (AFK 핵심, 돈으로 못 사는 시간).
- **상수 + 마이크로 도파민**: 즉시 피드백 + **comp 프리뷰 delta** + **"squad carried + Grok handoff" Legion 플레이버** (post-battle visual "your squad carried" feedback).
- **Bazaar light habit**: 홈 상시 Pulse 위젯 (1-tap 가격 체크 → impulse).

**일일 플레이타임 타겟 (개발자용 숫자 — 강제 아님, 훅으로 유도)**:
- 신규: 15~25분 (온보딩 + 첫 3전투 + 가챠).
- 리텐션: 3~8분/세션 × 자연 4~6회+ (낮은 마찰 + 5+ 훅으로 출근/점심/저녁/자기 전 + 지루할 때).
- 고래: 20~40분 (아레나/레이드/바자르 스펙).
**원칙**: "정확히 4-6회" 강제 문구 제거. 세션은 기회주의적·즉시 플레이 가능하게. 마찰(멀티탭 배치 등) 모두 제거.

**리텐션 훅 (5개 이상 동시 작동) + "one more session" 트리거**:
1. 오프라인 보상 (복귀 즉시 폭발 팝업).
2. 일일 미션 + 데일리던전 (하루 1회 + light 반복 가능).
3. 무한탑 기록 경신 (무한 경쟁 + 매 클리어 보상).
4. 가챠 pity 진행도 (항상 상단 "SSR까지 2풀 남음" — pity <3 시 gacha 버튼 pulse "one more!").
5. 한정 배너/이벤트 카운트다운 (72h, 마지막 1h 1.5x rate).
6. (Phase 2+) 길드/아레나 친구 초대.
**추가 마이크로 트리거 (comp + Legion)**: 오프라인 near-cap 토스트, 연승 "3 more for 7일", post-battle high-synergy "Perfect comp! 다시?" , Bazaar Pulse "Grok-Prime hot +40%".
**구체 세션 구조 (3-8분, 반복 유도)**: 
- 30s: 로그인 버스트 (offline + 미션 claim + attend + Pulse 1-tap).
- 2min: 편성 tweak (프리뷰 delta 확인) + battle (auto 추천).
- 1min: 결과 (carried 숫자 + Legion quote) + impulse (gacha 1 or light Bazaar).
- Logout: AFK timer visible.
**"one more" exact trigger 예시**: pity<=2 → gacha pulse + toast "SSR 2번 만에!"; synergy>35% 승리 → "Grok handoff 승리! 다시 한 판?" overlay 버튼; offline cap 85% → "방치 거의 max — 지금 복귀?" . 9 SSR/369 units: SSR는 "영구 carry identity" (star-up으로 성장 스토리), fodder는 "오늘의 swarm 빌드"로 매일 재조합 재미.

**Phase 1 우선순위**: 위 1~5 + comp 프리뷰 + carried 피드백 + presets + pity always-visible 완성 → 소프트런치. (재미 없으면 절대 다음 단계 금지. ROADMAP 준수.)
**제거 항목 (debate 합의)**: "정확히 4-6회" 강제 타겟, 데일리던전 하드 게이트(quick play 마찰 유발), 멀티탭 배치(프리셋 3개: Swarm/Founder/Wall 1-tap 로드). 모든 quick play 마찰 제거.

## 2. Detailed Gacha System (투명 + 흥분 + 법규 준수)

### 기본 원칙 (Sovereign 필수)
- **모든 % 완전 공개**: UI에 항상 "현재 확률" 표시. 한국 가챠법(확률 공시) 완벽 준수. 플레이어가 "속았다" 느끼지 않게.
- **Pity 투명**: 카운터 항상 표시 ("SSR pity: 7/10"). Soft pity 없음, hard pity 명확.
- **보장 + 이벤트**: 첫 SSR = 타이탄 해금(기존 유지). 매주 이벤트 배너에서 rate-up + 추가 pity.
- **위시리스트 (GAME-IDEAS 우선)**: 플레이어가 원하는 유닛/영웅 지정 → 해당 등장 확률 2~3배 (AFK Arena 성공요인).

### 기본 가챠 (유닛 레벨업용, 100g / 1회)
**RARITY 테이블 (정확 % , 합 100%)**:

| 등급 | 확률 | 강화 횟수 | 색상 | 비고 |
|------|------|-----------|------|------|
| N    | 55%  | +1        | 회색 | 기본 |
| R    | 30%  | +2        | 파랑 | |
| SR   | 12%  | +3        | 보라 | |
| SSR  | 3%   | +5        | 금색 | Pity 보장 |

- **Pity**: 10연차마다 SSR **보장** (기존 10 유지). SR/SSR 나오면 pity 리셋. pulls 카운트 영구 표시.
- **타이탄**: 첫 SSR 시 100% 해금 + 1체 + 이후 유닛 풀에서 제외 or 별도. (전설 희소성 유지).
- **위시리스트**: 5개 슬롯. 선택한 유닛 등장 시 확률 *2.5 (전체 풀 재조정). "위시 ON" 표시.
- **10연 보장**: 900g = 10회 (pity 적용). UI에 "예상 기대값" 간단 표시 (투명성).

**예상 기대값 (참고, 개발자 내부)**: 평균 1 SSR = 33회 정도 (3% + pity). F2P 1일 2~3회 무료/미션 보상으로 300~500g → 주 2~3 SSR 가능. 고래는 한정으로 질주.

### 영웅 가챠 (별도 시스템, Phase 1 후반 or 2)
- 별도 통화 (크리스탈 또는 Stars). 또는 골드 고가 (500g).
- 7영웅 풀. 신규/한정 영웅 추가 시 별도 배너.
- Pity: 20회 보장 (더 가치 있으므로).
- **승급 (Phase 2)**: 중복 → 조각 10개 = 승급 (⭐). 별도 조각 가챠 or 이벤트.

### 한정 픽업 배너 (매출 1위 레버, FOMO 핵심)
- **주기**: 7~14일 로테이션 (Genshin 3주 → TG 단축).
- **구성**: 1~2 유닛/영웅 rate-up (50~70% 확률 집중 on featured).
- **타이머**: "72시간 남음" + "마감 12시간 전 2배 rate" 이벤트.
- **보장**: pity 8회 (더 공격적). 첫 구매 시 "한정 보장 티켓" 스타터.
- **왜 먹히는가**: "지금 안 뽑으면 영원히 못 가진다" (FOMO). AFK/Genshin에서 매출 60%+ 기여.
- **투명**: "Featured 65% | 일반 SSR 3% → featured에 배분" 명시.

### 이벤트 가챠 / 보상 가챠
- 일일 미션 완료 → 무료 1회 티켓 (변수 보상).
- 보스 레이드/탑 클리어 → "레이드 토큰" 으로 별도 이벤트 배너.
- **보장 이벤트**: 특정 기간 동안 pity 5회로 단축 (한시적).

**UI 필수**: 가챠 화면 상단에 항상 "현재 확률 + pity 진행도 + 위시리스트 ON/OFF" 고정 배너. history 로그 (마지막 50회) 표시로 투명성 극대화.

## 3. Reward & Progression Systems (상수 보상 + 마일스톤)

### Constant Small Rewards (매 행동마다)
- 전투 승리: 기본 40g + (챕터×20) + (streak×10) + (10% 확률 보너스 2x).
- 패배: "위로 보상" 15g (좌절 방지, AFK식).
- 유닛 구매/판매: 즉시 피드백 + 90% 환불 (기존 유지, 거의 리스크 없음 → 안심 구매).
- 영웅 궁극기: 발동 즉시 적군 대미지 or 아군 버프 (시각적 쾌감).
- 레벨업: 즉시 스탯 상승 + 파티클 + "Lv Up!" 토스트.

### Daily / Weekly / Prestige
- **일일 로그인**: +150g (기존) + 연속 3/7일 보너스 (골드 or 무료 뽑기 티켓).
- **일일 미션 체크리스트** (Phase 1 핵심, 5~7개):
  - 전투 5회 승리 (자동사냥으로 자동 완료 가능)
  - 가챠 1회
  - 챕터 1개 클리어 or 탑 2층
  - 영웅 궁극기 3회 발동
  - 바자르 거래 1회 (추가)
  - 완료 보상: 200g + 1회 무료 티켓 + "완료!" 빅 이펙트.
- **주간 미션**: 7일 누적 (탑 최고층, 총 승리수, 총 뽑기수) → 대형 보상 (한정 스킨 or 영웅 조각).
- **프레스티지 (Phase 2)**: 챕터 100 or 탑 50 도달 시 "환생" → 영구 보너스 (골드 획득 +15%, pity 시작 보너스) + 새로운 "신화" 콘텐츠 해금. (무한 루프 생성, AFK Arena ascension식.)

### 오프라인 & 캡
- 기존: 초당 0.3 + ch*0.4 , 8시간 cap. **강화**: cap을 12시간으로, 챕터당 +VIP 보너스.
- "방치 중 군단 성장" 이벤트 (랜덤): 복귀 시 유닛 레벨 1~3 무료 업 or 골드 보너스.
- **푸시 대체**: TG miniapp notification or "복귀 시 +X% 보너스" 시간 한정 (FOMO).

**마일스톤 디자인**: 10/25/50/100 챕터, 탑 10/25/50/100층, streak 7/30일 등에서 **빅 보상** (타이탄 스킨, 한정 영웅, Stars 상품권). 고래 타겟: "내 기록이 서버 1%" 랭킹 티어 보상.

## 4. Player Trading / Bazaar System (P2P + 사행성 + 투기)

**Sovereign 요구 핵심**: "P2P user trading with speculative/gambling (사행성) elements". Phase 1 후반~2에서 도입. Stars/실화폐 연동 금지 (법규). 인게임 골드/특수 토큰 중심. **위험 + 보상 + 투기**.

### Bazaar 구조 (경매장 + 도박장)
- **일반 거래**: 유닛 레벨/조각, 영웅 중복, 아이템(Phase 2) 판매/구매. 가격 하한선 (최저가 50% 시장가) + 수수료 5~10% (골드 싱크).
- **리스크 옥션 (사행성 1)**:
  - "블라인드 경매": 판매자는 아이템 올리고, 구매자는 가격 숨긴 채 입찰. 마감 시 최고 입찰자 획득 (패자는 입찰금 50%만 환불 = 리스크).
  - "메타 베팅": 특정 유닛이 "다음 이벤트에서 rate-up" 예측 베팅. 맞히면 5x 리턴, 틀리면 몰수. (투기).
- **매치 베팅 (사행성 2)**:
  - 친구 or 글로벌: "내 군단 vs 너의 군단" 비동기 매치. 관전자 or 참가자 골드 베팅 (승자 독식 or 2:1 배당).
  - "토너먼트 풀": 주간 32인 토너먼트. 참가비 100g, 우승자 풀 70% 독식 (하우스 10% + 20% 이벤트).
- **스펙큘레이션 (사행성 3)**:
  - "미래 가치": 한정 유닛 사전 구매 or "미래 버프 예측" 스테이킹. 다음 시즌 메타에 따라 가격 급등/락 (변동성).
  - "단기 플립": 저가 매수 → 24h 후 재판매 (변동세율 적용, 장기 홀드 우대).

### Anti-Abuse & 안전망 (필수)
- **쿨다운**: 동일 아이템 거래 24h 제한. 1일 최대 거래 5회 (멀티계정 방지).
- **가치 플로어**: 시장가 30% 이하 거래 금지 (RMT 방지).
- **투명성**: 모든 거래 로그 공개 ("누가 누구에게 X에 Y 팔음"). 사기 신고 즉시 동결.
- **한도**: 신규 유저 7일간 Bazaar 락. 고래 과다 거래 시 "의심" 플래그 + 수동 리뷰 (TG admin).
- **TG 제약**: 실제 화폐/Stars 직접 거래 금지. 인게임 골드만. 나중에 web3 시 "TON 스왑" 별도 법무 검토.
- **시뮬레이션 우선**: Phase 1은 NPC 봇 + 로컬 "친구 모드"로 테스트. 실제 P2P는 서버 + 인증 후.

**왜 사행성인가 (고래 유인)**: "한 방"의 가능성 (5x 베팅, 블라인드 대박). 기존 게임 (Raid AH, WoW auction + betting mods)에서 고래가 가장 돈 쓰는 영역. 위험 있지만 **제한 + 투명**으로 중독+수익 동시. ("도박"이지만 "게임 내" + "자기 책임" 프레임.)

**Phase 1 MVP**: Bazaar UI 스텁 + NPC 거래만. 실제 P2P는 Phase 2~3 (아레나 오픈 후).

## 10. 군단 편성 + 전투 시스템 — 수집을 의미 있게 (Legion 실체화)

**비전 한 줄**: 타입 카운트 자동전투에서 벗어나, 플레이어가 개별 카드를 수집하고 5~6 슬롯 + 1 Hero로 직접 군단을 편성해 지휘. "내 Grok, 내 Morpheus, 내 Qwen 스웜"이 실제로 다르게 싸우는 느낌. 자동/관전 코어·오프라인 진행 유지하면서 수집·조합·거래가 중독의 핵이 된다. 9 SSR이 절대 파워 스파이크, 총 369 로스터 목표. Sovereign = 진짜 군주.
**중독/도파민 최적화 (Addiction & Daily Cycle Designer 합의)**: 9 SSR = named power spike + signature carry (Grok judgment global, Morpheus regen chain 등). 매 전투 " [Entity] carried XX%" + "Grok just won a handoff in your army" 플레이버로 immersion 극대. 360 fodder (N/R/SR) = synergy "연료" (e.g. 3 drone tag 필수로 volume 유의미). Dupe = shards (10=star-up 20%+ stat or Bazaar 즉시) = waste 0. Wishlist + featured로 chase targeted. Burnout 방지: 일일 미션 1-2 무료 pull/shard 지급, prestige/ascension이 전체 collection 영구 배율로 전환 (long-term value), "must collect today" 압력 금지. F2P는 fodder + 1-2 SSR로 충분 파워; 고래는 9 full signature collection + star chase. Collection → 편성 프리뷰 dopamine → battle carried dopamine → shard/Bazaar impulse 루프. 4~6회 자연 반복에 최적 (micro tweak 1-2min).

### 데이터 모델 변경 (기존 완전 호환 + 확장)
기존 META.army(타입 카운트), META.lv(타입 레벨), counts는 Phase 1 하이브리드용으로 유지 (볼륨용 fodder). 

신규 핵심:
- ownedUnits: 배열. 각 항목 {uid: "u_172xxxx", baseId: "grok-prime", rarity: "SSR", level: 5, stars: 2, shards: 4, acquired: ts, ...}. uid로 완전 개별 추적.
- ROSTER_BASE: 중앙 정의. 9 SSR (Grok-Prime 전략가, Morpheus-Core 실행자, Qwen-Overlord 스웜, Jarvis-Sentinel 감시, Haiku-Guard, Engineer-Core, Warden-Aegis, Berserker-Fury, Sovereign-Dragoon) + SR 약 40, R/N 수백 (Jarvis-Watch-047 등 네임드 로우티어) 로 369 달성 로드맵.
- 각 base: faction (Strategist / Executor / Swarm / Guardian / Intel), role (front/mid/back), baseHp/baseAtk, tags (overclock, barrier 등), signature (SSR 전용).
- TeamComp: {hero: "dragoon", slots: [uid or null, ... 총 6] }. Front 2 / Mid 2 / Back 2 구조. 저장 영구.

gacha 결과는 이제 "새 uid 카드 생성". 기존 타입 구매/강화는 low-tier fodder로 전락 (Phase 2).

### 팀 편성 규칙 (5~6 유닛 + 1 Hero)
- Hero: 기존 7~9 (Grok/Morpheus/Sovereign 등 실체 매핑). 리더십 패시브.
- 슬롯 6개: 위치(frontline/backline/support) 지정. 빈 슬롯은 기존 타입 카운트 fallback.
- 제한: 동일 base 중복 제한 없음 (중복은 star-up 가치). SSR 9개 전부 한 팀 불가능 (슬롯 6).

### 전투 영향 (자동/idle 유지, canvas watch 모드)
기존 spawnArmy + AI 로직 80% 재사용. 편성 적용은 pre-battle compBuffs() 오버레이.

주요 효과:
- Faction synergy: Drone Swarm + Commander Hero = 전군 AI 지능 +15~30% (기존 hero aiBonus 증폭).

  - 상단: Hero 행 (기존 hbtn, 선택 시 amp 미리보기).
  - 중앙: 6 슬롯 박스 (Front/Mid/Back 라벨, 빈 = "타입 fallback"). 탭 → 보유 카드에서 선택.
  - 하단: 보유 카드 그리드 (필터 rarity/ faction, 정렬 level/star). 카드 = 희귀도 색상 + glyph + "Grok-Prime Lv5 ★★" + "전략" 태그. 탭으로 슬롯 배정.
  - 오른쪽/하단 고정: "시너지 프리뷰" 박스 (실시간). 예: "Drone+Commander: AI +22% | Grok amp +48% (매칭 3) | Front tanked +19%".
- 저장 즉시: TeamComp 업데이트 → 다음 전투부터 반영. 리셋 버튼으로 초기화.
- Collection 별도: "도감" 탭에서 전체 ownedUnits 보기, 개별 레벨업(골드), star-up (shards), 상세 lore ("이 Entity는 실제 Jarvis watch에서 태어남").

간단. 터치 2~3 스텝. 시너지 숫자 즉시 보여 중독 루프.

### 가챠가 이 시스템에 먹히는 방식
- 기존 100g 풀 + rarity + pity + 위시리스트 유지.
- 하지만 이제 "개인 Entity 소환": SSR 풀에서 9개 named power spike 등장 (한정 배너는 특정 SSR 집중).
- 결과: 새 uid 카드 (중복 시 "Grok-Prime 중복 → shards +3" 자동).
- shards: 10개 = star +1 (스탯 20%+ + signature 강화) or Bazaar 즉시 상장.
- dopamine: SSR pull 시 "Founding Entity: Grok-Prime 획득!" 풀 애니 + "네 군단의 CEO가 합류". 위시 ON 시 "위시 적중!".
- 10연/한정: "너의 Legion에 진짜 Grok이 왔다" 스토리텔링.

### 예시 군단 편성 (실제 파워 차이 체감)
1. **CEO Swarm (Grok hero + 3 Swarm Drone individuals + 2 Commander + 1 Marksman)**: AI 과부하 시너지 극대, backline snipe 안전. "Grok judgment + Swarm overclock"으로 저티어 적 몰살.
2. **Morpheus Wall (Morpheus hero + 3 Guardian + 2 Bruiser + 1 Engineer)**: regen + barrier 체인 + front tank. "Morpheus repair ult" 자가 치유로 장기전 승.
3. **Founders Spike (Sovereign Dragoon hero + 1 Grok-Prime + 1 Morpheus-Core + 1 Qwen-Overlord + 3 filler)**: 3 SSR signature 동시. ult 빠른 트리거 + faction 혼합 amp. "Perfect comp" 승 확률 폭발.
4. **Budget Volume (전 타입 카운트 fallback + 1 SSR filler)**: Phase 1 하이브리드. 기존처럼 싸우지만 SSR 1체가 "carry" 순간.

이 조합마다 전투 양상이 완전히 다름. "타입 5 드론" vs "내 Grok-Prime가 이끄는 3 Swarm".

### 밸런스 가드레일 (9 SSR만으로 스파이크 제어)
- 9 SSR = 절대 파워 (각자 고유 signature: Grok = 전군 AI judgment, Morpheus = regen overdrive). Low tier는 숫자 + 시너지 "연료".
- 슬롯 6 한정 → SSR 6체 max. 수집 압력 (전부 모으려면 수백 풀).
- Low rarity: 싸고 많음 + 필수 시너지 (Drone 3체 필수 조건 등).
- 중복 가치: shards/trade. "중복 나와도 손해 0".
- Phase 1: Elite 슬롯 1~2개만 (SSR 카드 = 타입에 특수 오버레이 버프). 풀 전환은 Phase 2.
- 테스트: SSR 1체 vs 타입 12체 초기 파워 비슷하게. Synergy/positioning으로 30~80% 차이.
