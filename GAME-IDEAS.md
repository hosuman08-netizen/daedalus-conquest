## 핵심 발견 (데이터)
1. **순수 tap-to-earn은 죽어간다.** Notcoin 팀도 "모델 끝났다" 인정 — 깊이 없고 멀티계정 어뷰징·토큰 번아웃. → **게임성 + 장기 리텐션이 생존 조건.** (우리는 이미 오토배틀+육성 깊이 보유 = 좋은 포지션)
2. **AFK Arena 성공공식** (Lilith를 억만장자로): 다팩션 영웅 + **위시리스트 가챠**(픽업 영향력) + **진형/시너지 전투** + 방대한 콘텐츠(길드전·타워·원정·시간의균열) + **시간 기반 진행**(AFK 보상이 핵심, 돈으로 못 사는 시간).
3. **매출 1위 동력 = 신규 캐릭터 한정 배너(FOMO).** 천장 필수. 사라지기 전에 뽑게 만드는 게 돈.
4. **고래 1~2%가 매출 대부분.** 그들에겐 상한이 없다 → 최상위 상품(VIP·한정) 깊이 필요.
5. **리텐션 핵심 = 길드/연합 + 아레나.** 아레나는 "일일 5회 무료 + 젬으로 추가" 구조가 표준.
6. **배틀패스**는 좌절 적고 호평 — 특히 **재구매형(CoD식: 벌어서 다시 삼)**.

## 🔥 2026 벤치마크: 중독·보상·투명 가챠·P2P 사행성 핵심 메카닉 9선 (고래/리텐션 무자비 분석)

**추출 기준**: 실제 매출 1위 동력 (FOMO + 안전망 + 변동성 + 소셜 압력). 데이터 기반, Telegram/Stars 친화 + AI Legion 테마 연계 가능성 우선. 출처: 공식 rates + 커뮤니티 분석 (Genshin pity 0.6% base/90 hard + 50/50, RAID 0.5% mercy ramp, Duolingo loss aversion, Skyblock AH flipping).

1. **Genshin/Honkai: Soft Pity Ramp + Hard Pity + 50/50 투명 공시**  
   Base 0.6% 5★, 74부터 ramp (~6%씩 ↑), 90 hard guarantee. Featured 50/50 (lose 시 다음 100% rate-up). Consolidated avg 1.6%.  
   **왜 중독?** "진행감" (pity 카운터) + "안전망"으로 ragequit ↓, soft pity 구간 "one more pull" 도박 심리 극대. 50/50는 리스크/릴리프 루프. 고래: constellation (중복) chase.  
   **투명도**: 인게임 exact % + pity 표시 필수 (법규+신뢰).

2. **Limited Banners + FOMO 타이머 (Genshin/RAID/Epic7 공통)**  
   7~14일 회전 한정 픽업 + "이번에 못 뽑으면 몇 달/영원히".  
   **왜?** 즉시 과금 압력. 고래 매출 60-80% 여기서 나옴. Variable reward (배너마다 다름) + scarcity.

3. **AFK Arena/RAID: Wishlist + Mercy Pity + Guild/Synergy**  
   Wishlist: 플레이어 지정 유닛 확률 미세 부스트. Mercy (RAID Ancient 0.5% base, 200풀 후 +0.5%/pull ramp). Guild wars, faction synergy.  
   **왜?** Agency (내가 뽑는 느낌) + 오프라인 AFK (시간=자산) + 소셜 의무 (길드 일일). 오프라인 진행 = "로그아웃해도 성장" 상수 보상.

4. **Duolingo: Streak + Loss Aversion + Variable Daily Rewards**  
   Streak 불 유지 = loss aversion (잃는 고통 > 얻는 기쁨 2배). Freeze 아이템, 랜덤 XP/배지/리그 변동 보상. 알림 습관화.  
   **왜 리텐션 최강?** 7일 streak 유저 next-day retention 2.4x. Telegram notification + mini streak 완벽 매치.

5. **Hypixel Skyblock / PoE: Player-Driven AH + Flipping + Speculative Economy**  
   Auction House/Bazaar: 실시간 가격 변동, under-cut sniping, "buy low sell high" flipping. 시장 이벤트(신규 콘텐츠)로 가격 폭등/폭락.  
   **왜 사행성 중독?** 스킬+운+타이밍 도박. "오늘 2배 수익" 변동성 도파민. 커뮤니티 가격 토론 = 소셜 루프. PoE trade는 게임의 반.

6. **Cookie Clicker / Clicker Heroes: Prestige Reset + Exponential Idle Growth**  
   Reset으로 영구 % multipliers (1% CpS / level, cubic scaling). Automation layers + "numbers go up" 루프.  
   **왜?** 영구 진행감 + "조금만 더 모으면 reset으로 비약". Idle 게임의 핵심 중독 루프. Daedalus 이미 오토배틀+오프라인 있음 → prestige만 추가하면 즉시 증폭.

7. **Clash of Clans: Builder Offline Progress + Seasons + Clan Wars**  
   Builder 타이머 (오프라인 병렬 성장), 시즌 태스크/보상 트랙, 클랜전 (15v15 사회 경쟁 + 보상).  
   **왜?** "빌더 바쁨" = 시간 압력 (현질 skip). 시즌 FOMO + 클랜 소속감.

8. **Epic Seven / Summoners War: Arena PvP + Gear/Rune Variance Farming**  
   비동기/실시간 아레나 + 깊은 gear 최적화 (runes = high variance roll = 도박). Awakening/승급 이벤트.  
   **왜 고래?** PvP 순위 = status + 보상. Gear farming "just one more roll" 루프.

9. **Battle Pass (Re-purchasable) + Event Cycles (모든 히트작)**  
   무료+프리미엄 트랙, 시즌마다 리셋 구매 유도. 이벤트 빈도: 주 1회 회전 대형 + 일일 소형.  
   **왜?** 예측 가능한 과금 + FOMO. 고래 재구매 유도. 좌절 적고 지속 매출.

**공통 고래/리텐션 진실**: 1-2% 유저가 50-80% 매출. FOMO (한정) + Safety (pity) + Status (PvP/Collection) + Gambling (variance + flipping) + Social (guild/arena). Tap-to-earn 죽은 이유: 깊이/중독 루프 부재.
