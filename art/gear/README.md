# Gear images (g1-g120)

Drop generated PNGs here as g{id}.png (e.g. g1.png for 강철 검).

gearArt() in game.js will automatically prefer the PNG in:
- char panel equipped slots (cp-gear)
- inventory cards (.gear-card)
- page-gear list (.gitem)
- codex (gdex)

If no PNG, falls back to upgraded synth (veins, floating shards, heavy layered shadows, volumetric feel — already "간지" level).

Prompts: See ../GEAR-ART-PROMPTS.md (research-backed from HSR/Genshin/Epic7 + Warframe/Destiny/AC + Diablo/PoE etc. for cool premium look, 모방 OK 카피 NO).

Quota 때문에 여기서 직접 gen 실패. 프롬프트 복사해서 외부 툴로 gen 후 drop.
