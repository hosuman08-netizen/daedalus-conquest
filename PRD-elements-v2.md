# PRD (v2 기둥): 5속성 시스템 + 소울 승격 (Elemental Pillar)

> 작성: 2026-06-16 · 작성자: CPO(트리니티) · 상태: **초안 / 출시 후 v2 (출시 전 착수 금지)**
> 전략 검토: CEO(자비스/그록) — 전투 상성 휠은 비전·밸런스 근간이라 CEO 승인 필요.

## 0. 한 줄
유닛에 **5속성(불·물·토·빛·어둠) 정체성 + 상성 휠**을 부여하고, 죽어있던 **소울을 5속성 소울로 승격**해 — "같은 유닛도 속성별로 다 모은다"는 **수집 욕구 = 가챠 LTV 2차 로켓**을 만든다. 출시(MVP) 이후 첫 메이저 업데이트의 핵심 기둥.

## 1. 문제 / 왜 (지금이 아니라 *출시 후*)
- **고통(출시 후 단계)**: 환생 루프가 D7~D14 리텐션을 살리고 결제가 켜져도, **"왜 *더* 뽑아야 하나"의 깊이**가 약하다. 유닛 종류가 한정 → 가챠가 금방 "다 모음"에 도달 → 과금 천장이 낮음.
- **속성이 푸는 것**: 같은 유닛이라도 **속성별 변형 = 수집 대상 5배**, 상성 휠 = "내 팀에 빠진 속성" 구매 동기. 콘텐츠도 속성별로 확장(같은 엔진 재활용 — 신규 적 제작 최소).
- **왜 *지금 안* 하나**: 전투 상성 재설계 + 전 유닛 속성 배정 + UI + 밸런스 = 수 주짜리. 출시 막는 두 벽(환생·결제)과 자원 충돌 → **출시 후로 못박음.** (sim·문서 설계는 지금 가능, 코드는 출시 후.)

## 2. 목표 (성공 기준 — 측정 가능)
- **North Star 보강**: "유닛당 보유 속성 수"(수집 깊이) ↑ → ARPU·D30 리텐션 ↑.
- **1차 지표**: D30 리텐션, ARPPU(과금자 객단가), 1인당 보유 속성-유닛 조합 수, 속성 가챠 배너 전환율.
- **가드레일**: 신규 유저가 속성 휠에 압도당해 초반 이탈하면 안 됨(초반엔 휠 숨김, ch10+ 점진 공개) / 기존 보유 유닛이 속성 도입으로 *약화*되면 안 됨(소급 손해 금지) / 화폐 난립으로 UI 혼란 없을 것.

## 3. 사용자 / JTBD
- **대상**: 출시 후 잔존한 중·과금 유저 + 환생 루프 1회+ 경험자(메타 진행에 맛 들린 층).
- **Job**: "내 군단을 *완성*하고 싶다 — 같은 영웅도 속성별로 모으고, 상성으로 어려운 전투를 *머리로* 뚫고 싶다. 빠진 속성 한 칸이 자꾸 눈에 밟힌다(수집 강박)."

## 4. 핵심 설계 — 속성 휠
**5속성**: 🔥불(Fire) · 💧물(Water) · 🪨토(Earth) · ✨빛(Light) · 🌑어둠(Dark)
**상성 휠** (클래식 검증형 — SW/에픽세븐 모델):
- **3원소 순환**: 🔥불 → 🪨토 → 💧물 → 🔥불 (이긴 쪽이 +피해·+치명, 진 쪽은 빗맞음 확률).
- **빛 ↔ 어둠 상호 카운터**: 서로에게 +피해(둘 다 강), 3원소엔 중립.
- **속성 우위 효과**: 우위 공격 시 **+30% 피해 + 치명률 ↑**, 열위 시 **약공격(빗맞음) 확률**. → 전투가 가위바위보 머리싸움이 됨(깊이).

## 5. 소울 승격 — 화폐 역할 분리 (★중요)
난립 방지를 위해 **두 메타 화폐의 차선을 명확히 분리**:
- **에테르(Ether)** = *수평* 영구 파워. 환생 루프, 속성 무관, 전군 배율. (환생 PRD 소관)
- **속성 소울(5종)** = *수직* 속성별 유닛 투자. 해당 **속성 유닛의 각성/승급 재료.** 🔥불의 소울로 불 유닛만 각성.
→ 에테르=넓게, 소울=깊게. 겹치지 않음. 기존 단일 소울은 **마이그레이션**(보유 소울 → 분배 또는 '무지개 소울'로 아무 속성 교환).

## 6. 핵심 사용자 흐름
1. ch10+ 도달 → 속성 휠 튜토리얼 1컷 공개(그 전엔 숨김 — 신규 보호).
2. 전투 패배/고전 → "이 스테이지는 🔥불 우위" 힌트 → "불 유닛이 없네" 자각.
3. 속성 배너에서 불 유닛 가챠 → 속성 소울로 각성 → 우위로 스테이지 돌파(머리로 이김 도파민).
4. "이 영웅, 물 버전도 있네?" → 같은 영웅 속성 컬렉션 욕구 → 재가챠(루프).

## 7. 범위 — v2 MVP (속성 기둥 1차)
- [ ] 5속성 정의 + 상성 휠 전투 로직(우위 +30%·치명, 열위 빗맞음).
- [ ] 전 유닛에 속성 1개 배정(기존 로스터 소급 — 약화 없이).
- [ ] 단일 소울 → 5속성 소울 마이그레이션 + 무지개 소울(교환).
- [ ] 속성 소울 = 속성 유닛 각성/승급 재료로 연결.
- [ ] 속성 휠 UI(전투 중 우위/열위 표시) + ch10+ 점진 공개.

## 8. 비범위 — v2.1 이후로 미룸 (한 번에 다 X)
- ❌ 속성별 일일 던전(요일 로테이션) — v2.1.
- ❌ 속성 한정 가챠 배너·속성 픽업 — v2.1(수익화 2차).
- ❌ 속성별 진영/세계관 리스킨 — 후순위.
- ❌ 같은 영웅의 속성 변형 아트 5종 — 비용 큼, 단계적(처음엔 색 팔레트 변형).

## 9. 지표·가정·리스크
- **가장 위험한 가정**: "유저가 *같은 영웅의 다른 속성*을 새 수집 대상으로 인지하고 다시 뽑는다." → **싼 검증**: 출시 후 코호트에 속성 휠만 먼저(가챠 변형 없이) → 속성-우위 전투 사용률·만족 측정 후 속성 배너 투입.
- **리스크**:
  - 전 유닛 속성 배정 = 밸런스 전면 재점검(balance-sim에 속성 차원 추가 필요).
  - 화폐 9종 난립 위험 → §5 역할 분리·소울 마이그레이션으로 차단(신규 화폐는 소울 *분화*지 *추가* 아님).
  - 신규 유저 인지 과부하 → ch10+ 점진 공개·초반 휠 숨김.
  - 소급 손해 불만 → 기존 유닛 속성 배정은 *추가*만(스탯 하향 절대 금지).

## 10. 시퀀싱 (군단 합의용)
1. **출시(MVP)**: 환생 + 결제 + 버프 가시화. (이 PRD 손도 안 댐)
2. **v2 (출시 후 1st major)**: 이 PRD §7 = 속성 휠 + 소울 승격.
3. **v2.1**: §8 = 속성 던전 + 속성 배너(수익화 2차 로켓).
→ ②페어 버프(조합 버프 PRD)도 v2 트랙에 동승.
