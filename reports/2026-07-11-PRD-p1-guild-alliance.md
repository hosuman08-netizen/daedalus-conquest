# PRD — p1 다이달로스 · 길드/연맹 (Legion Clans)
**작성**: 트리니티(CPO) · 2026-07-11 · 근거: 경쟁 딥다이브(Whiteout $3.4B·Hero Wars $1.5B·Coin Master의 심장 = 사회적 락인)

## 왜 (문제/기회)
- **p1 최대 공백 = 사회적 락인 0.** 딥다이브 승리공식 #4: "연맹은 선택 아님"이 리텐션 심장. 개인 이탈을 **동료 압박**으로 막는 게 방치/가챠 최강 리텐션 레버.
- 유입0 국면에서도 유효: 초기 100명이 **서로 물어뜯고 서로 붙잡게**(coldstart-founding-100 결합). 습격(Raid) 엔진 이미 있음 → 길드가 그걸 사회화.
- Legion hegemony 도트린(clan UGC)과 정확히 일치.

## 무엇 (North Star & 성공지표)
- **North Star 기여**: 주간 신규활성화 + **D7/D30 리텐션**(길드 가입자 vs 미가입자 리텐션 델타 = 핵심 검증).
- 가드레일: 강요·죄책감(confirmshaming) 금지. 미가입 페널티 아닌 **가입 보너스**로 프레임(파놉티콘 Mode A).
- 가설: 길드 가입 시 D7 리텐션 +15%p 이상(Duolingo/Whiteout 벤치 근거).

## MVP 범위 (가장 위험한 가정부터, 싸게)
**가장 위험한 가정**: "저인구(N~수십)에서도 길드가 작동하나?" → **MVP는 봇+실유저 혼합 소형 길드(정원 10)**로 검증. 실유저 부족분은 fictional NPC 멤버(정직 라벨: "AI 군단원")로 채워 사회적 존재감 확보.

### MVP-1 (되돌림가능·클라단독, trinity→grok 구현)
1. **길드 생성/가입** — 이름·엠블럼(기존 아트). 정원 10. 로컬+KV(uid). 가입 즉시 **가입 보너스**(골드/젬).
2. **길드 기여 리더보드** — 주간 기여도(전투승리·습격·소환 카운트 합산). 사회비교. **실카운트만**(가짜 금지).
3. **길드 공동 목표(온도계)** — 주간 집단 목표(예: 합산 100승) 달성 시 전원 보상. 협력압력. 실집계.
4. **길드 채팅(간이)** — 프리셋 이모트/문구(자유입력은 모더레이션 리스크 → v2). 소속감.

### MVP-2 (구조·neo 승인 게이트)
5. **길드전(Guild Raid)** — 기존 Raid 엔진을 길드 vs 길드/보스로. 주간 케이던스.
6. **초대-게이트 티탄 습격** — 길드원 초대 시 강력 습격 해금(K-factor). ⚠️start_param 귀속 백엔드=neo.

## 안 할 것 (이번엔)
- 자유 텍스트 채팅(모더레이션 부담) · 실시간 동기 PvP(인프라) · 크로스서버 병합(인구 확보 후) · 길드 화폐 별도 경제.

## 미꾸라지/합법
- 사회적 증거는 **실카운트만**(가짜 시청자수류 금지=FTC). NPC 멤버는 "AI 군단원" 정직 라벨.
- 강요·죄책감 금지 → 가입 **보너스** 프레임. 파놉티콘 Mode A 감사 게이트.

## 오너십/시퀀스
- **PRD·설계·지표 = 트리니티** / **구현 = grok(또는 trinity 클라단독분)** / **KV·귀속 백엔드·배포 = neo 승인**.
- 순서: MVP-1(1→2→3→4, 되돌림가능 연속출하) → 검증(리텐션 델타) → MVP-2(길드전·초대게이트, neo 게이트).
- 계측: 길드_생성/가입/기여/목표달성/채팅 이벤트 배선(오라클). instrument-before-ignite 게이트.
