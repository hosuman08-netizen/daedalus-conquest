# SPEC: 도파민 모먼트 맵 — "세지는 게 보이는 게임"

> CPO 트리니티 · 2026-06-17 · 구현 우선순위 가이드 (→모피어스)
> 정체성: **"다이달로스 = 매 순간 내 군단이 세지는 게 *보이는* 게임."**
> 원리: 도파민 = ①numbers go up(숫자 상승 시각화) ②즉각 체감(강화→바로 더 빨리 녹임) ③예측+보상(천장→SSR) ④절정(환생=영구배율로 폭살)

## 핵심 규칙
1. **첫 10분에 "팡" 5~6번** — 모먼트 간격이 길면 이탈. 도파민은 빈도.
2. **모든 성장은 "보여야" 한다** — 강화/가챠/환생 시 *전력 숫자 롤업 + 전후 비교*(이전엔 막히던 적을 이제 순삭). 보이지 않는 성장 = 0.
3. **연출 투자는 절정에 몰아라** — 첫 SSR·첫 환생에 최대 스펙터클.

## 우선순위 (도파민 ROI 순)
1. **🔴 첫 환생 절정 연출** (★★★★★, 시그니처) — 정체성의 핵심. "녹이는" 쾌감을 최대 과장. 여기가 약하면 정체성이 안 섬.
2. **🔴 첫 SSR 스펙터클** (★★★★) — 첫인상·수익화 훅. 금빛 폭발+암전+사운드.
3. **🟠 첫 강화 "세짐 체감"** (★★★) — 전력 롤업 + 전후 전투 대비(적 더 빨리 녹음)를 명확히.
4. **🟠 전력 누적 가시화** (15분, ❌없음) — "시작보다 N배" 한 줄로 누적 성장 체감.
