# SPEC: 골드 뽑기 + 캐릭 제물 분해 → 소울 루프 (군주 발안)

> CPO 트리니티 · 2026-06-17 · 구현: 모피어스(COO)
> 목적: ①골드 싱크 ②소울 수급처 추가 ③무과금 성장 루프 ④수집·분해·각성 (재미 있는 피드백).
> **사상 위치**: 이 루프는 *무과금 자원순환* — hooks(near-miss/FOMO/scarcity)는 **가챠·한정배너·prestige window에 집중**하고, 여기선 깔끔한 성취 피드백으로 무과금 잔존을 떠받친다(무과금 플라이휠이 과금 hooks의 토대). 미꾸라지: 확률·수치 정확 공개.

## 무과금 루프 (이걸 만든다)
전투 → 골드 → **🪙골드 뽑기** → N/R 양산 → **제물 분해 → 소울** → **각성 강화** → 더 강해짐(세지는 게 보임)

## ♻️ 골드 복리 플라이휠 (군주 핵심 통찰)
골드는 `dividendGold`(= legionPower×0.6)로 **전투마다 복리로 계속 지급**된다 → 골드뽑기 연료가 마르지 않음.
→ **자가발전 플라이휠**: 강함↑ → 골드배당↑ → 골드뽑기↑ → 분해↑ → 소울↑ → 각성↑ → **강함↑** (선순환).
**유저가 강해질수록 사이클이 더 빨리 돈다** = 무과금 리텐션 엔진. 이게 이 시스템의 진짜 동력.
- **가드(CPO)**: star3 게이트가 소울 인플레 차단(이미). 골드배당량 vs 골드뽑기 비용은 플레이테스트로 균형 — 골드가 과하면 뽑기비용↑로 조정해 플라이휠 속도 제어.

## 1. 🪙 골드 단차 뽑기 (신규)
- **비용**: 200 골드 / 1뽑. (10연 = 1,800골드, 10% 할인 + SR↑ 1보장)
- **확률표**: N 70% · R 25% · SR 5% · **SSR 0%**
- **천장 없음** (양산용, SSR 안 나오므로 불필요)
- ⚠️ **SSR 0% 절대 준수** — 골드는 전투로 흔하니 SSR 나오면 💎젬 뽑기(=매출)가 죽는다. 골드 뽑기는 N/R/SR까지만.
- 젬 뽑기(💎8, N60/R25/SR13/SSR2)와 **공존**: 골드=양산/분해재료, 젬=SSR 노림 프리미엄.

## ⭐ SSR 중복 처리 — 유저 선택 (군주 결정 2026-06-18)
SSR도 중복 뜸. SSR 9개+업데이트로 늘어나니 "안 쓰는 SSR은 큰 보상으로, 키울 SSR은 깊이로" — **선택은 유저에게.**
- **일괄분해(N/R/SR)는 SSR 제외 유지**(실수 방지). SSR 중복은 **개별 선택 액션**(상세창/도감) + 확인 모달.
- 두 선택지:
  | 선택 | 보상 |
  |---|---|
  | 🌟 **돌파** | 같은 SSR 캐릭 **각성(✦) 또는 승급(★) +1** (영구 깊이 육성). 무과금 SSR 육성 가속 |
  | ⚗️ **분해** | **소울 80**(희소 반영, SR15의 5배+ = "보상 크게"). 안 쓰는 SSR 정리 |
- 가드: 분해는 확인 모달 필수(되돌릴 수 없음). 편성중·잠금 SSR 제외.

## 2. 캐릭 제물 분해 → 소울
| 등급 | 분해 소울 | 비고 |
|---|---|---|
| N | **2** | 양산 fodder, 주 분해 대상 |
| R | **5** | |
| SR | **15** | 분해 시 확인 모달 |
| SSR | **60** | 분해 시 확인 모달 + 기본 잠금 |

- **중복 개수 표시** (군주): 도감/인벤 캐릭 카드에 **보유 수량 ×N 뱃지**. 같은 캐릭 여러 장 = 한눈에.
- **일괄 변환** (군주): **"중복 전부 소울로" 버튼 1개** — 보유 2장째부터(첫 1장은 보존) 한 번에 분해 → 합산 소울 획득. 인벤 정리 + 보상 (압박 없는 재미 피드백). FOMO/중독 framing 금지.
- **실수 방지**: SR/SSR은 확인 필수, SSR/편성중/잠금 캐릭은 일괄변환에서 **제외**(첫 1장도 보존). 잠금 토글 제공.
- 분해/일괄변환 시 **소울 획득 연출**(숫자 팝 + 사운드 = 도파민). 일괄은 "+N 소울!" 큰 팝.

## 3. 소울 용도 (확장 — 군주 "소울로 할 수 있는 거 만들면 되지")
분해로 소울 수급이 늘므로 **쓸 곳(싱크)도 늘려야** 인플레 방지 + 도파민. 우선순위:
1. **각성** (기존 메인): 30·60·120·240·480 소울 ×5단계, **star 3 요구**. 영구 스탯.
2. **🔮 소울 상점** (신규, 무과금 자원 순환):
   | 교환 | 소울 비용 | 의미 |
   |---|---|---|
   | 골드 ×5,000 | 20 | 골드↔소울 순환(골드 뽑기 재투자) |
   | 강화석/승급 재료 | 30~50 | star3 게이트 돌파 보조 → 각성 진입 |
   | 💎 젬 소량(×50) | 80 | 무과금이 소울로 프리미엄 가챠 향함(완만) |
   | (확장) SR 조각 → 확정 교환 | 150 | 무과금 수집 천장 — 후속 |
- 소울 수급 = 보스전(2+ch/6) + **분해**(주력) + 일괄변환.
- 가드: 소울→젬 교환은 **완만하게**(80소울=50젬). 너무 후하면 젬 매출 잠식.

## 4. 균형 검증 (수치 근거)
- 골드 뽑기 **기대 소울** = 0.70×2 + 0.25×5 + 0.05×15 = **3.4 소울/뽑**
- 200골드/뽑, 전투 1판 보상 ≈ 200~440골드 → **전투 1판당 ~3.4 소울**
- 각성 1단계(30소울) ≈ **9뽑 ≈ 9전투**. 5단계 총(930소울) ≈ 274뽑 = **장기 목표** ✓
- **인플레 자연 게이트**: 각성은 **star3 필수** → 소울 남아도 star3(승급) 없으면 정체. 소울 인플레로 후반 콘텐츠 소진되는 것 방지 ✓
- **플레이테스트 조정점**: 골드 뽑기 비용(200) · 분해 소울량 · SR 5% 는 출발값. 무과금 각성 속도가 너무 빠르면 비용↑ or 분해량↓.

## 5. UI (모피어스 구현)

- 도감/캐릭 인벤: **분해 액션** + 중복 자동분해 토글 + 잠금 + 확인 모달.
